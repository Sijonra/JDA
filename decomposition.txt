- [ ] Проверить верстку в различных браузерах (Chrome, Firefox, Safari, Edge)

### Документация
- [ ] Обновить README.md с инструкциями по установке и запуску проекта

### Git Flow
- [ ] Создать ветку для задачи "Верстка основного контента"
- [ ] Создать ветку для задачи "Добавление анимаций"

- [ ] Произвести слияние Pull Request после проверки

### Коммиты
- [ ] Выполнить коммиты согласно спецификации conventional commits

### Финальная проверка
- [ ] Провести финальную проверку верстки перед завершением проекта
